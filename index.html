如何能超越现在的你....

发表于 2020-02-25

前言

最近在看一套视频教程，在课程的开始导言中，讲师一席话给了我动力，听完之后，感觉真的是满满的正能量，让我感慨，让我沉思，从他的讲话中，我get到了我所需要的点，那就是”决心+坚持” = ?😁😁😁
常言道：不忘初心，方得始终，初心易得，始终难守。人生是一场漫长而又艰辛的接力赛，有时领先，有时候落后，但只要在路上，就有机会…

没事的时候，就少看一些小视频，去少玩一些游戏，去少聊一些八卦新闻，放下手中的手机，打开我们的编辑器。去那里静静地敲一个小时的代码。我相信，你会有一个不错的这种学习的状态，只要有决心的人，才会去成就一番事业（不喜勿喷）。还有就是，只有学会了去平静的对待生活中冷眼与嘲笑的人。才会去有所成就。只有说，学会了品味孤独，并且呢，去享受一个人安静时光的这种人。他才能够学到知识与技能。大家都知道，学会了自律呢!（自律的人很可怕….^_^），这样才会给你更大的这种自由。实现一个财务的自由(这也许是一个美好的愿望，哈哈哈)，这也是想去告诉大家的。所谓的舒服，那是不存在的。我们来到了这个世界，就是为了去吃苦，如果说，你的家庭是一个富裕的家庭。如果说，你是一个没有这种上进心和追求的人，但是我告诉你的是。我看到了N年对生活没有追求的人，他现在生活的就是非常的糟糕。但是呢，我同样也看到了那些努力了N年的人，现在呢，过上了自己想要的生活。所以说，你想要成为什么样子的人，是和你的决心相关的……，你想要过什么样纸的生活，是由你现在的状态决定的。所以说，但行好事，莫问前程”, 永远不要和别人比，只要今天的自己，比昨天好，就OK….👋👋 大家都努力吧！！！

Linux 常用命令整理

发表于 2020-02-05 更新于 2020-02-09 分类于 Linux

前言

利用宅在家里的机会，把自己经常用的一些Linux基础命令整理一下，方便自己日后查阅，哈哈，好记性不如烂笔头……^_^

ls 该命令通常用来查看文件夹中的所有文件;
ls -a 该命令通常用来查看文件夹中的隐藏文件;
man 查看帮助信息（例如：man ls）;
echo 该命令在终端输出字符串;
date 查看当前系统时间;
reboot 命令用于重启系统;
Poweroff 命令用于关闭系统;
wget 在终端中下载网络文件;
ps 命令用于查看系统中的进程状态
top 用于动态地监视进程活动与系统负载等信息
free 显示内存使用量信息
uptime 用于查看系统的负载信息
kill 命令用于终止某个指定PID的服务进程(例如：kill -1892)
killall 终止某个指定名称的服务所对应的全部进程
ifconfig 获取网卡配置与网络状态等信息
uname 用于查看系统内核与系统版本等信息
last 查看所有系统的登陆记录
history 用于显示历史执行过的命令
who 查看当前登入主机的用户终端信息
pwd 显示用户当前所处的工作目录
cd 切换工作路径
cat 查看纯文本文件（内容较少）
more 用于查看纯文本文件(内容较多)，注：只能向前移动查看；
less 查看文件
head 查看纯文本文档的前N行;
tail 查看纯文本文档的后N行或持续刷新内容;
diff 用于比较多个文本文件的差异;
touch 创建文件；
mkdir 创建文件目录；
mv 剪切文件或将文件重命名；
rm 用于删除文件或目录；
file 用来识别文件类型，也可用来辨别一些文件的编码格式
find 命令用于按照指定条件来查找文件 (例如： find -name mysql)
tar 用于对文件进行打包压缩或解压
cp命令用于复制文件或目录;
grep 在文本中执行关键词搜索，并显示匹配的结果;
wc命令用于统计指定文本的行数、字数、字节数;
