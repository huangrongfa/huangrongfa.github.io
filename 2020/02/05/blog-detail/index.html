网站部署上线流程

  • 一.开发完成本地代码提交到github
  • 1.git status
  • 2.git add .
  • 3.git commit -m ‘’
  • 4.git push origin marter
  • 二.部署上线
  • 1.在Mac中输入 sudo -i
  • 2.ssh root@ 服务器IP（例如：root# ssh root@...*）
  • 3.输入服务器密码
  • 4.登陆成功
  • 5.git pull拉取仓库代码
  • 6.在项目中设置可以执行的shell脚本
  • 7.登陆代码仓库，在settings中设置webhooks比如：http://...*:端口号/
  • 8.进入项目目录例如webserve目录，然后执行./webserve
  • 9.上线成功
