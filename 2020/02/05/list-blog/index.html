Linux 常用命令大全

  • ls命令通常用来查看文件夹中的所有文件
  • ll命令用来查看文件夹中的所有文件(包含隐藏文件)
  • alias命令可以用来设置别名，通常alias可以这样设置 《新命令=’原命令 选项/参数’》
  • cat /etc/passwd 看当前操作系统有哪些用户
  • sudo su - 切换到超级管理员
  • history 命令可以查看使用当前使用过的Linux命令
  • pwd 查看当前路径
  • df - h 查看当前系统文件使用情况
  • du -sh * 查看当前文件的占用空间大小
  • env 查看当前环境变量( 可以立即执行这个命令让配置文件立即生效source ~/.bash_profile）
  • ps -ef |grep 查看当前进程（例如：ps -ef |grep mysql)
  • netstat -tunpl |grep 8080 查看端口的使用情况
  • chmod 777 修改文件权限
  • chown root:root 设置**的所属者和所属组都是root
  • whereis 可以用来查找文件
  • Which 查看可执行文件的位置
  • man 查看帮助信息（长格式和短格式）
  • echo 该命令在终端输出字符串或变量提取后的值
  • date 查看当前系统时间
  • reboot 命令用于重启系统
  • Poweroff 命令用于关闭系统
  • wget 在终端中下载网络文件
  • ps 命令用于查看系统中的进程状态
  • top 用于动态地监视进程活动与系统负载等信息
  • free 显示内存使用量信息
  • uptime 用于查看系统的负载信息
  • kill 命令用于终止某个指定PID的服务进程
  • killall 终止某个指定名称的服务所对应的全部进程
  • ifconfig 获取网卡配置与网络状态等信息
  • uname 用于查看系统内核与系统版本等信息
  • last 查看所有系统的登陆记录
  • history 用于显示历史执行过的命令
  • who 查看当前登入主机的用户终端信息
  • pwd 显示用户当前所处的工作目录
  • cd 切换工作路径
  • ls 显示目录中的文件信息
  • cat 查看纯文本文件（内容较少）
  • more 用于查看纯文本文件(内容较多)
  • less
  • head 查看纯文本文档的前N行
  • tail 查看纯文本文档的后N行或持续刷新内容
  • diff 用于比较多个文本文件的差异
  • touch 创建文件
  • mkdir 创建文件目录
  • mv 剪切文件或将文件重命名
  • rm 用于删除文件或目录
  • file 用来识别文件类型，也可用来辨别一些文件的编码格式
  • find 命令用于按照指定条件来查找文件
  • tar 用于对文件进行打包压缩或解压
  • cp命令用于复制文件或目录
  • grep 在文本中执行关键词搜索，并显示匹配的结果
  • wc命令用于统计指定文本的行数、字数、字节数
